## Task 1: Perform Basic Mathematical Operations

### Description
This program performs basic mathematical operations on two numbers entered by the user.

### Functionality
- Accepts **two numbers** as input from the user.
- Performs the following operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Displays the result of each operation clearly on the screen.

### Sample Output
Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2.0

### Concepts Used
- User input (`input()`)
- Arithmetic operators (`+`, `-`, `*`, `/`)
- Print statements

## Task 2: Create a Personalized Greeting

### Description
This program generates a personalized greeting message using the user's name.

### Functionality
- Takes the user's **first name** and **last name** as input.
- Combines both inputs to form a **full name**.
- Prints a personalized greeting message.

### Sample Output
Hello, John Doe! Welcome to Python program

### Concepts Used
- String input
- String concatenation
- Output formatting
